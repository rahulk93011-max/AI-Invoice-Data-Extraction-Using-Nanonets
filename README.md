# AI Invoice Data Extraction Using Nanonets

## Project Overview

This project demonstrates how Artificial Intelligence can automatically extract invoice information from PDF documents using Nanonets OCR.

The system extracts supplier details, purchaser details, invoice information, payment terms, item descriptions, quantities, prices, and totals without manual data entry.

## Business Problem

Organizations receive hundreds of invoices every month. Manual invoice processing is:

- Time consuming
- Error prone
- Costly
- Difficult to scale

AI-powered invoice extraction automates the process and improves accuracy.

## Objective

To automate invoice data extraction from purchase invoices using Nanonets AI OCR.

## Tools Used

- Nanonets OCR
- PDF Invoices
- Microsoft Excel
